## 03 . 주석 / 파일, 클래스 명명 방법

## 주석

```js
/**
 * @Class			클래스명
 * @Date			날짜 기입
 * @Author			작성자
 * @Brief			클래스 용도
 */
```


## 파일 / 클래스 명명 방법
클래스와 파일이름은 같아야 한다.
 - 2.x 버전 : 파일이름은 <b class="blue">소문자</b>, 클래스이름은 <b class="red">대문자</b>로 시작
 - 3.x 버전 : 파일이름과 클래스이름이 대소문자까지 같아야 한다.
